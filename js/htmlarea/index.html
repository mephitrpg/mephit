Note

  • Compatibile con Explorer e Mozilla.
  • I pulsanti sono personalizzabili editando l'array "toolbar" che si trova nella riga 120 del file htmlarea.js.
  • Per rimuovere pulsanti basta usare gli inline comments (il commento "/**/") all'interno della dichiarazione dell'array "toolbar" (è possibile commentanre parte di una riga).
  • Per disporre i pulsanti su una o più righe, editare l'array "toolbar". Esso è un array di array. Ad ogni elemento dell'array padre (delimitato dalle parentesi quadre) corrisponde una riga.
  • I pulsanti della toolbar non vanno a capo. Settando la larghezza della textarea avanzata ad un valore minore di quella della toolbar, l'area di testo editabile verrà ridotta, ma le dimensioni della textarea avanzata rimarranno quelle della toolbar.
  • Per dichiarare un semplice CSS per la textarea editare la riga 97 del file htmlarea.js.
    Esempio:
    this.pageStyle = "body{background-color:#FFFFFF;font:10pt verdana;color:#000000;}a{color:880000;}";

Bugs

  • Per qualche motivo ancora non chiaro, la funzione "initEditor()" dà problemi se viene richiamata all'interno della tabella che contiene la textarea o comunque annidata in alcuni tags. In caso di errore spostare il richiamo della funizone dopo la chiusura di tags o tabelle varie.
